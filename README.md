# 福田林校请假管理系统

## 项目结构

```
FT-SchoolFlow5/
├── frontend/          # 前端项目 (Vue 3 + TypeScript + Element Plus)
├── backend/           # 后端项目 (待开发)
└── docs/              # 项目文档
```

## 前端项目

前端项目位于 `frontend/` 目录下，使用 Vue 3 + TypeScript + Element Plus 技术栈。

### 启动前端项目

```bash
cd frontend
npm install
npm run dev
```
